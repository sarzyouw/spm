# Sistem Informasi Penjaminan Mutu Internal 

Sistem Informasi Penjaminan Mutu (SPM) Politeknik STMI Jakarta untuk digitalisasi siklus PPEPP. Terintegrasi database MySQL untuk manajemen dokumen audit, repositori SOP, dan monitoring capaian standar mutu program studi. Dirancang untuk efisiensi audit internal (AMI) secara real-time, menggantikan proses manual menjadi terpusat dan akuntabel.

# Fitur Utama
* Dashboard Admin / Sekretaris SPM : Kelola Hak Akses dan Dokumen.
* Dashboard Ketua SPM : Validasi dokumen.
* Pengunjung / Guest : dapat melihat dokumen terkait Penjaminan Mutu STMI.

# Teknologi
* Framework: Laravel
* Database: MySQL
* Frontend: Bootstrap 5 & Inter Fonts
